# 八角笼助手

八角笼助手是一款面向《英雄联盟》国服自定义海克斯大乱斗玩法的本地辅助工具。

如果你经常和路人一起玩八角笼玩法，它可以帮你：

- 记录玩家的正向标签和负向标签
- 在再次遇到这些玩家时进行提醒
- 在对局结束、有人秒退、离开房间后，快速回顾并打标
